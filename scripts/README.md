# Scripts Directory

This directory contains standalone scripts that can be executed independently, including both shell scripts and Python tools.

## Active Maintenance

### [binance_vision_api_aws_s3](./binance_vision_api_aws_s3)

This is the main directory that is actively maintained. It contains scripts for interfacing with Binance Vision API and AWS S3 services.

## Deprecated Directories

The following directories are not maintained and may be deprecated or removed in future updates:

- [arrow_cache](./arrow_cache)
- [dev](./dev)
- [dsm_demo](./dsm_demo)
- [funding_rate_downloader](./funding_rate_downloader)

## Future Development

For experimental features and upcoming projects, development will initially take place in the [`playground/`](../playground) directory. Once these projects mature and are deemed stable, they may be moved here to the `scripts/` directory for production use.
